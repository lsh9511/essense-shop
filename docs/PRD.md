# PRD (Product Requirements Document)
# ESSENCE - 미니멀 프리미엄 편집샵

## 1. 프로젝트 개요

### 1.1 프로젝트 목표
30대를 위한 미니멀 프리미엄 편집샵 플랫폼을 구축하여, 신진 브랜드 큐레이션을 통해 차별화된 쇼핑 경험을 제공하고 실제 수익을 창출합니다.

### 1.2 타겟 사용자

#### 고객 (Customer)
- **연령**: 28-38세
- **직업**: 직장인, 프리랜서
- **소득**: 중상위층 (연봉 4,000만원 이상)
- **특징**:
  - 품질과 디자인을 중시
  - 브랜드 스토리에 관심
  - 합리적 소비 추구
  - 미니멀리즘 라이프스타일

#### 어드민 (Admin)
- 쇼핑몰 운영자
- 상품 등록, 주문 관리, 재고 관리 담당

#### 브랜드 파트너 (Partner)
- 신진 브랜드 디자이너
- 제품 공급 및 브랜드 스토리 제공

## 2. 주요 기능 명세

### 2.1 고객 기능 (Customer Features)

#### 2.1.1 회원 관리

**회원가입**
- 이메일 회원가입
- 소셜 로그인 (카카오, 네이버, 구글)
- 필수 정보: 이메일, 비밀번호, 이름, 전화번호
- 선택 정보: 생년월일, 성별
- 마케팅 수신 동의 (선택)

**로그인**
- 이메일 + 비밀번호
- 소셜 로그인
- 자동 로그인 (7일간 유지)
- 비밀번호 찾기 (이메일 인증)

**마이페이지**
- 프로필 정보 수정
- 비밀번호 변경
- 배송지 관리 (기본 배송지, 추가 배송지)
- 회원 탈퇴

#### 2.1.2 상품 조회

**상품 목록 페이지**
- **필터링**:
  - 카테고리 (상의, 하의, 아우터, 신발, 액세서리)
  - 브랜드
  - 가격대 (10만원 이하, 10-20만원, 20-30만원, 30만원 이상)
  - 색상
  - 사이즈
  - 정렬: 신상품순, 인기순, 가격 낮은순/높은순

- **상품 카드 표시 정보**:
  - 상품 이미지 (썸네일)
  - 브랜드명
  - 상품명
  - 가격
  - 할인율 (있을 경우)
  - 품절 표시
  - 찜 버튼 (하트)

**상품 상세 페이지**
- **상품 정보**:
  - 고화질 이미지 (3-5장, 확대 가능)
  - 브랜드명 (클릭 시 브랜드 페이지 이동)
  - 상품명
  - 가격, 할인가
  - 상품 설명 (소재, 특징)
  - 사이즈 가이드
  - 착용 모델 정보 (키, 몸무게, 착용 사이즈)

- **옵션 선택**:
  - 사이즈 선택 (S, M, L, XL 등)
  - 컬러 선택 (이미지로 표시)
  - 수량 선택
  - 재고 표시 (품절/재고 5개 이하 알림)

- **액션 버튼**:
  - 장바구니 담기
  - 바로 구매
  - 찜하기

- **추가 정보 탭**:
  - 상세 설명
  - 사이즈 정보
  - 배송 안내
  - 교환/반품 안내
  - 리뷰 (별점, 사진 리뷰)

**브랜드 페이지**
- 브랜드 소개 (철학, 스토리)
- 브랜드 이미지/로고
- 해당 브랜드 상품 목록
- 브랜드 SNS 링크

**검색**
- 통합 검색 (상품명, 브랜드명)
- 자동완성
- 최근 검색어
- 인기 검색어

#### 2.1.3 장바구니

- 장바구니 상품 목록
- 옵션 변경 (사이즈, 컬러, 수량)
- 개별 상품 삭제
- 선택 상품 삭제
- 찜하기 이동
- 총 상품 금액, 배송비, 최종 결제 금액 표시
- 쿠폰 적용 (할인 금액 표시)

#### 2.1.4 주문/결제

**주문서 작성**
- 주문 상품 목록 (옵션 확인)
- 배송지 입력
  - 기존 배송지 선택
  - 새 배송지 입력 (우편번호 검색)
  - 배송 메모 (선택)
- 주문자 정보 (이름, 전화번호, 이메일)
- 쿠폰 선택
- 결제 수단 선택
  - 신용카드
  - 계좌이체
  - 간편결제 (토스, 네이버페이, 카카오페이)
- 총 결제 금액 확인

**결제**
- Toss Payments 연동
- 결제 성공/실패 처리
- 주문 완료 페이지 (주문번호, 예상 배송일)

**주문 내역**
- 주문 목록 (날짜별 정렬)
- 주문 상태: 입금 대기, 결제 완료, 상품 준비중, 배송중, 배송 완료, 구매 확정
- 주문 상세 정보
  - 주문 번호
  - 주문 날짜
  - 상품 정보
  - 배송지 정보
  - 결제 정보
  - 송장 번호 (배송중일 때)
- 배송 조회 (스마트택배 API 연동)
- 구매 확정 버튼
- 리뷰 작성 버튼

#### 2.1.5 찜하기/위시리스트

- 찜한 상품 목록
- 상품 삭제
- 장바구니 담기
- 품절 상품 알림 신청

#### 2.1.6 리뷰

**리뷰 작성**
- 별점 (1-5점)
- 텍스트 리뷰
- 사진 업로드 (최대 5장)
- 착용 정보 (키, 몸무게, 착용 사이즈)
- 사이즈 만족도 (작아요, 적당해요, 커요)

**리뷰 조회**
- 상품 상세 페이지에서 리뷰 목록
- 별점 평균, 리뷰 개수
- 사진 리뷰 필터
- 최신순, 별점순 정렬

**마이페이지 리뷰 관리**
- 작성한 리뷰 목록
- 리뷰 수정/삭제

#### 2.1.7 교환/반품

- 교환/반품 신청
  - 주문 선택
  - 사유 선택 (단순 변심, 사이즈 불만족, 불량 등)
  - 상세 사유 입력
  - 사진 첨부 (불량 시)
- 교환/반품 내역 조회
- 상태: 신청, 승인, 수거중, 수거 완료, 환불 완료

### 2.2 어드민 기능 (Admin Features)

#### 2.2.1 대시보드

**주요 지표**
- 오늘의 주문 건수
- 오늘의 매출
- 이번 달 매출
- 이번 달 순이익
- 미처리 주문 건수
- 재고 알림 (품절 임박 상품)

**그래프**
- 일별 매출 그래프 (최근 30일)
- 카테고리별 매출 비율 (파이 차트)
- 인기 상품 Top 10

#### 2.2.2 상품 관리

**상품 등록**
- 상품명
- 브랜드 선택
- 카테고리 선택
- 가격, 할인가
- 상품 설명 (에디터)
- 상품 이미지 업로드 (드래그 앤 드롭, 최대 10장)
- 썸네일 선택
- 옵션 설정
  - 사이즈 옵션 추가 (S, M, L 등)
  - 컬러 옵션 추가 (색상명, 색상 이미지)
  - 각 옵션별 재고 수량
  - 각 옵션별 추가 가격 (선택 사항)
- 상세 정보
  - 소재
  - 제조국
  - 세탁 방법
  - 사이즈 상세 (가로, 세로, 어깨너비 등)
- 착용 모델 정보
- 배송 정보 (배송비, 배송 기간)
- 상태 (판매중, 품절, 판매 중지)

**상품 목록**
- 상품 목록 테이블
  - 썸네일
  - 상품명
  - 브랜드
  - 가격
  - 총 재고
  - 상태
  - 등록일
  - 수정/삭제 버튼
- 검색 (상품명, 브랜드)
- 필터 (카테고리, 브랜드, 상태)
- 엑셀 다운로드

**상품 수정**
- 등록과 동일한 폼
- 이미지 추가/삭제
- 옵션 추가/삭제/수정

**상품 삭제**
- 소프트 삭제 (DB에는 남김)
- 주문 내역이 있는 상품은 삭제 불가 경고

#### 2.2.3 재고 관리

**재고 현황**
- 상품별 재고 목록
  - 상품명
  - 옵션 (사이즈, 컬러)
  - 현재 재고
  - 안전 재고 (최소 유지 수량)
  - 재고 상태 (정상, 부족, 품절)
- 재고 부족 알림 (안전 재고 이하)
- 재고 엑셀 다운로드

**재고 입고**
- 상품 선택
- 옵션 선택
- 입고 수량
- 입고 날짜
- 메모 (공급처, 입고 비용 등)

**재고 조정**
- 재고 수량 직접 수정
- 조정 사유 입력 (분실, 불량, 재고 정리 등)

**재고 이력**
- 입고/출고 이력 조회
- 날짜, 상품, 수량, 사유

#### 2.2.4 주문 관리

**주문 목록**
- 주문 목록 테이블
  - 주문번호
  - 주문일시
  - 주문자명
  - 상품명 (여러 개일 경우 "외 N건")
  - 결제 금액
  - 주문 상태
  - 배송 상태
  - 상세보기 버튼
- 필터
  - 주문 상태 (전체, 입금 대기, 결제 완료, 상품 준비중, 배송중, 배송 완료, 구매 확정)
  - 날짜 범위
  - 검색 (주문번호, 주문자명, 전화번호)
- 엑셀 다운로드

**주문 상세**
- 주문 정보
  - 주문번호
  - 주문일시
  - 주문 상태
- 주문자 정보
  - 이름, 전화번호, 이메일
- 배송지 정보
  - 수령인명, 전화번호
  - 주소
  - 배송 메모
- 상품 정보
  - 상품명, 옵션, 수량, 가격
- 결제 정보
  - 상품 금액
  - 배송비
  - 할인 금액 (쿠폰)
  - 최종 결제 금액
  - 결제 수단
  - 결제일시

**주문 상태 변경**
- 입금 확인 → 결제 완료
- 결제 완료 → 상품 준비중
- 상품 준비중 → 배송중 (송장번호 입력 필수)
- 배송중 → 배송 완료 (자동 또는 수동)
- 배송 완료 → 구매 확정 (고객 또는 7일 후 자동)

**송장 입력**
- 택배사 선택
- 송장번호 입력
- 입력 시 자동으로 "배송중" 상태로 변경
- 고객에게 배송 시작 알림 발송

**주문 취소**
- 취소 사유 선택
- 결제 취소 처리 (Toss Payments 환불 API)
- 재고 복구

#### 2.2.5 교환/반품 관리

**교환/반품 신청 목록**
- 신청 목록 테이블
  - 신청일
  - 주문번호
  - 신청자명
  - 상품명
  - 사유
  - 상태 (신청, 승인, 수거중, 수거 완료, 환불 완료)

**교환/반품 처리**
- 신청 상세 확인
  - 신청 사유
  - 첨부 사진
- 승인/거부
- 수거 택배 발송
- 수거 완료 확인
- 환불 처리 (Toss Payments 환불 API)
- 재고 복구

#### 2.2.6 브랜드 관리

**브랜드 등록**
- 브랜드명
- 브랜드 로고 이미지
- 브랜드 소개 (에디터)
- 브랜드 철학/스토리
- SNS 링크 (인스타그램, 홈페이지 등)
- 담당자 정보 (이름, 연락처, 이메일)
- 정산 정보 (정산 비율, 계좌 정보)

**브랜드 목록**
- 브랜드 목록 테이블
- 수정/삭제

**브랜드 정산**
- 브랜드별 판매 내역
- 기간 선택
- 총 판매액, 정산 금액
- 정산 내역 엑셀 다운로드

#### 2.2.7 고객 관리

**고객 목록**
- 고객 목록 테이블
  - 이름
  - 이메일
  - 전화번호
  - 가입일
  - 총 주문 건수
  - 총 주문 금액
  - 회원 등급 (일반, VIP)
- 검색 (이름, 이메일, 전화번호)

**고객 상세**
- 고객 정보
- 주문 내역
- 리뷰 내역
- 메모 (어드민 전용)

#### 2.2.8 쿠폰 관리

**쿠폰 생성**
- 쿠폰명
- 할인 타입 (정액 할인, 정률 할인)
- 할인 금액/비율
- 최소 주문 금액
- 최대 할인 금액 (정률일 때)
- 사용 기간 (시작일, 종료일)
- 발급 대상 (전체, 특정 회원)
- 발급 수량 (무제한 or 제한)

**쿠폰 목록**
- 쿠폰 목록 테이블
- 사용 통계 (발급 수, 사용 수)

**쿠폰 발급**
- 전체 회원 발급
- 특정 회원 발급 (이메일로 선택)
- 쿠폰 코드 공개 (공개 URL)

#### 2.2.9 통계 및 분석

**매출 통계**
- 일별/주별/월별 매출 그래프
- 기간별 매출 비교
- 순이익 계산 (매출 - 원가 - 배송비 - 수수료)

**상품 통계**
- 인기 상품 Top 20
- 카테고리별 판매 비율
- 브랜드별 판매 비율
- 재고 회전율

**고객 통계**
- 신규 가입자 추이
- 재구매율
- 평균 구매 금액 (객단가)
- 회원 등급별 분포

**주문 통계**
- 주문 건수 추이
- 결제 수단별 비율
- 시간대별 주문 분포

**마케팅 통계**
- 쿠폰 사용률

## 3. 비기능 요구사항

### 3.1 성능
- **응답 시간**: 페이지 로딩 시간 2초 이내
- **이미지 최적화**: WebP 포맷, Lazy Loading
- **동시 접속자**: 최소 100명 동시 접속 처리

### 3.2 보안
- **HTTPS**: 전구간 HTTPS 적용
- **비밀번호**: bcrypt 해싱
- **JWT**: Access Token (1시간), Refresh Token (7일)
- **SQL Injection 방지**: SQLAlchemy ORM 사용
- **XSS 방지**: 입력값 sanitize
- **결제 정보**: PCI-DSS 준수 (Toss Payments 사용)

### 3.3 UX/UI
- **미니멀 디자인**: 깔끔하고 세련된 디자인
- **모바일 반응형**: 모바일 최적화 필수
- **로딩 인디케이터**: 로딩 중 스켈레톤 UI
- **에러 처리**: 친절한 에러 메시지

### 3.4 SEO
- **메타 태그**: 상품별 메타 태그 설정
- **사이트맵**: 자동 생성
- **구조화된 데이터**: Schema.org 적용 (Product, Review)

### 3.5 접근성
- **키보드 네비게이션**: Tab 키로 이동 가능
- **대체 텍스트**: 이미지 alt 속성
- **색 대비**: WCAG AA 수준
