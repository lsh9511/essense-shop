# ESSENCE (에센스)

> 30대를 위한 미니멀 프리미엄 편집샵

## 프로젝트 개요

**ESSENCE**는 본질에 집중하는 신진 브랜드 큐레이션 플랫폼입니다.
과도한 트렌드보다 시간이 지나도 변하지 않는 가치, 세련된 디자인과 우수한 품질을 중시하는 30대를 위한 온라인 편집샵입니다.

### 핵심 가치

- **본질 (Essence)**: 불필요한 것을 제거하고 본질에 집중
- **큐레이션 (Curation)**: 엄선된 신진 브랜드만 소개
- **품질 (Quality)**: 합리적 가격의 프리미엄 제품

## 비즈니스 모델

### 타겟 고객
- **연령**: 30대 (28-38세)
- **특징**: 품질과 디자인을 중시하는 실용주의자
- **소득**: 중상위층 (구매력 있음)
- **라이프스타일**: 미니멀리즘, 지속가능성 관심

### 소싱 전략
- **신진 브랜드 입점**: 독점 큐레이션으로 차별화
- **직접 재고 보유**: 품질 관리 및 빠른 배송
- **가격대**: 10만원~30만원 (프리미엄 합리가)
- **카테고리**: 의류, 신발, 액세서리

### 수익 구조
- **판매 마진**: 제품 판매 마진 (목표 40-50%)
- **입점 브랜드 관리**: 월 정액 또는 수수료 모델 (선택 사항)

## 주요 기능

### 1. 고객 기능

#### 쇼핑
- 상품 목록/상세 페이지 (고화질 이미지, 상세 설명)
- 사이즈/컬러 옵션 선택
- 장바구니 및 찜하기
- 주문 및 결제 (Toss Payments, 카드, 간편결제)
- 배송 조회

#### 회원 관리
- 회원가입/로그인 (이메일, 소셜 로그인)
- 마이페이지 (주문 내역, 배송 조회)
- 찜 리스트 관리
- 리뷰 작성

#### 브랜드 스토리
- 브랜드 페이지 (브랜드 철학, 스토리)
- 브랜드별 제품 모아보기

### 2. 어드민 기능

#### 상품 관리
- 상품 등록/수정/삭제
- 사이즈/컬러 옵션 관리
- 재고 관리 (입고, 출고, 재고 알림)
- 상품 이미지 업로드

#### 주문 관리
- 주문 목록 조회
- 주문 상태 변경 (입금 확인, 발송 처리, 배송 완료)
- 배송장 번호 입력
- 환불/교환 처리

#### 브랜드 관리
- 입점 브랜드 등록/관리
- 브랜드별 판매 통계
- 정산 관리

#### 통계 및 분석
- 매출 통계 (일/주/월별)
- 인기 상품 분석
- 재고 회전율
- 고객 구매 패턴 분석

### 3. 마케팅 기능

- 쿠폰/할인 코드 생성
- 이메일 마케팅 (신상품, 프로모션)
- 리뷰 이벤트
- 추천 상품 (개인화)

## 기술 스택

### Backend
- **FastAPI**: 고성능 Python 웹 프레임워크
- **SQLAlchemy**: ORM
- **PostgreSQL**: 메인 데이터베이스
- **Redis**: 세션, 장바구니 캐싱
- **Celery**: 비동기 작업 (이메일 발송, 통계 계산)

### Frontend
- **React**: UI 라이브러리
- **TypeScript**: 타입 안정성
- **TailwindCSS**: 스타일링 (미니멀 디자인)
- **React Query**: 서버 상태 관리

### 결제/배송
- **Toss Payments**: 결제 게이트웨이
- **스마트택배 API**: 배송 조회

### 인프라
- **Docker**: 컨테이너화
- **AWS/GCP**: 클라우드 배포 (선택 사항)
- **S3**: 상품 이미지 저장

## 차별화 포인트

1. **신진 브랜드 큐레이션**: 대형 플랫폼에 없는 독점 브랜드
2. **스토리텔링**: 브랜드의 철학과 스토리 강조
3. **30대 타겟팅**: 구매력 있는 특정 연령층 집중 공략
4. **미니멀 디자인**: 깔끔하고 세련된 UX/UI
5. **프리미엄 합리가**: 품질 대비 합리적 가격

## 예상 비용 및 수익

### 초기 투자 비용
- 재고 확보: 500만원 ~ 1,000만원 (10-20개 스타일)
- 개발 비용: 0원 (직접 개발)
- 마케팅 비용: 100만원 ~ 300만원 (초기 3개월)
- 기타 비용: 50만원 (사업자등록, 도메인, 호스팅)
- **총 예상 비용**: 650만원 ~ 1,350만원

### 수익 목표 (월 기준)
- **1개월차**: 매출 0원 (개발 기간)
- **2-3개월차**: 매출 100만원 ~ 300만원 (소프트 런칭)
- **4-6개월차**: 매출 500만원 ~ 1,000만원 (마케팅 본격화)
- **7-12개월차**: 매출 1,500만원 ~ 3,000만원 (안정화)

### BEP (손익분기점)
- 평균 마진 40% 가정
- 고정비 월 50만원 (호스팅, 마케팅 최소)
- **BEP 매출**: 약 125만원/월
- **예상 BEP 달성**: 3-4개월차

## 리스크 및 대응 전략

| 리스크 | 대응 전략 |
|--------|----------|
| 재고 부담 | 소량 입고 → 판매 확인 후 재입고 |
| 초기 트래픽 부족 | SNS 마케팅, 인플루언서 협업 |
| 경쟁사 (무신사, 29CM 등) | 신진 브랜드 독점 큐레이션으로 차별화 |
| 브랜드 확보 어려움 | 신진 디자이너 직접 발굴, 협업 제안 |
| 반품/교환 비용 | 명확한 사이즈 가이드, 고화질 이미지 제공 |

## 마케팅 전략

### 초기 (1-3개월)
- **인스타그램**: 브랜드 감성 콘텐츠 발행 (주 3-5회)
- **네이버 블로그**: 브랜드 스토리, 코디 팁 포스팅
- **인플루언서 협업**: 마이크로 인플루언서 (팔로워 1-5만)

### 중기 (4-6개월)
- **네이버 스마트스토어**: 네이버 검색 유입 활용
- **구글 광고**: 타겟 키워드 광고
- **콘텐츠 마케팅**: 패션 매거진 스타일 콘텐츠

### 장기 (7-12개월)
- **커뮤니티 구축**: 고객 리뷰, 스타일 공유
- **브랜드 콜라보**: 한정판 제품 출시
- **오프라인 팝업**: 브랜드 경험 제공

## 성공 지표 (KPI)

### 비즈니스 지표
- **월 매출**: 목표 1,000만원 (6개월 내)
- **객단가**: 목표 15만원 이상
- **재구매율**: 목표 30% 이상
- **재고 회전율**: 목표 분기당 3회 이상

### 마케팅 지표
- **웹사이트 방문자**: 월 5,000명 (6개월 내)
- **전환율**: 목표 2% 이상
- **SNS 팔로워**: 5,000명 (6개월 내)
- **고객 획득 비용 (CAC)**: 목표 3만원 이하

### 운영 지표
- **평균 배송 기간**: 2-3일
- **반품률**: 10% 이하
- **고객 만족도**: 4.5/5.0 이상
